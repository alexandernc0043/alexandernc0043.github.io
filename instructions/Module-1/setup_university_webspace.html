Step 1 - Set up your computer

  1. Enable showing file extensions on your system.
    • Windows Instructions:
      1. Open File Explorer (Shortcut: WIN + E)
      2. Click View
      3. Hover over Show
      4. Click File name Extensions
    • MacOS Instructions:
      1. Open Finder
      2. Click Finder on the Menu Bar (Top Left)
      3. Click Settings (Shortcut: CMD + ,)
      4. Navigate to Advanced
      5. Enable Show all filename extensions
  2. Install a Code Editor of your choosing. We recommend using Visual Studio Code (VS Code) which can be downloaded here.
    If you install Visual Studio Code please download the Live Server Extension from the Extensions Marketplace
  3. Install an FTP Client We recommend using FileZilla, which can be downloaded here.
  4. Install the University's VPN Client
    You can download from here
    Login:
    • Username: Niner_Net_username@charlotte.edu
    • Password: Niner_Net_password
    Replace the bolded parts with your university provided login
  6. Install GitHub Desktop
    You can download from here.
    You do not need to use GitHub Desktop and may use another Git Client or use the Git CLI. We recommend it as not everyone is familiar with git and CLI tools. But Instructions will assume that you installed it.

Step 2 - Setup GitHub

  1. Go to GitHub.com and sign in or sign up for an account.
  2. Navigate to the create repository button and create a repository.
  3. Name the repository as follows: YOUR_GITHUB_USERNAME.github.io If you do not name your repository exactly like that your webpage will not work.
  4. Open GitHub Desktop and clone your repository.
  5. Open your cloned repository in Visual Studio Code by clicking the button inside GitHub Desktop

Step 3 - Creating your first webpage.

  1. Inside Visual Studio Code create a new file called index.html.
  2. Using Visual Studio Code's Emmett tool type an ! and hit Tab or Enter on your keyboard.
  3. Your file should look like this now:
                        
<!DOCTYPE html>
    
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>

</body>
</html>

  4. Edit your title by editing the word Document inside this element <title>Document</title> with {YOUR_NAME} {Divider} Homepage

    Example(s): 
                        
<title>Alexander Prechtel | Homepage</title>
<title>Alexander Prechtel ~ Homepage</title>
<title>Alexander Prechtel - Homepage</title>

  5. Inside your body element add a Heading 1 element (h1) and put the following:

                        
<body>
    <h1>Welcome to my homepage!</h1>
</body>
  6. Now add an anchor element (``a``) after your heading 1 element that will take you to your course page. 
    <a href="itis3135">Click here to go to my ITIS-3135 course page!</a>

    Your body should now look like this: 
    
<body>
    <h1>Welcome to my homepage!</h1>
    <a href="itis3135">Click here to go to my ITIS-3135 coursepage!</a>
</body>

Step 4 - Creating your course page

  1. Inside your GITHUB_USER_NAME.github.io folder create a folder named itis3135
    Note: in the future your GITHUB_USER_NAME.github.io folder will be referred to as the "root" of your website
  2. inside your itis3135 folder create a new file called index.html
  3. Open your newly created index.html file inside your itis3135 folder.
  4. Copy the contents index.html inside the root of your website and paste it into your newly created index.html file inside your itis3135 folder.
  5. Remove the anchor element (a)

  6. Replace everywhere you have homepage with ITIS-3135 course page

    Your new index.html file should look like this now: 
                        
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Alexander Prechtel | ITIS-3135 Course page</title>
</head>
<body>
  <h1>Welcome to my ITIS-3135 course page!</h1>
</body>
</html>
