Understanding Functions in Kotlin

Kotlin functions are declared using the fun keyword. They are versatile and can be defined with parameters, return types, default values, and even as single-expression functions for concise code.

  • Function Declaration: Functions are declared with the fun keyword followed by the function name, a parameter list, and an optional return type.
  • Parameters and Return Types: You can define parameters with their types and specify a return type. If a function does not return a meaningful value, it returns Unit.
  • Default Arguments: Functions can provide default values for parameters, reducing the need for multiple overloads.
  • Single-Expression Functions: For simple operations, you can write functions as single expressions without curly braces or an explicit return statement.
  • Higher-Order Functions: Kotlin supports passing functions as arguments and returning them, allowing for functional programming techniques.

Kotlin's function features allow you to write clean, efficient, and expressive code that can handle a variety of programming tasks.

        
fun greet(name: String): String {
    return "Hello, $name!"
}

fun add(a: Int, b: Int): Int = a + b  // Single-expression function

fun printGreeting(name: String = "Kotlin User") {
    println(greet(name))
}
